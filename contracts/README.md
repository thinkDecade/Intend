### Token Mapping
- **iUSDT** — 1:1 sandbox equivalent of real USDT (Tether)
- **iXAUT** — 1:1 sandbox equivalent of Tether Gold (XAUT)

### Faucet
New users are automatically credited with 100,000 iUSDT + 100,000 iXAUT on both chains via `agent/faucet.js`.
