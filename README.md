# Intend

**Your money, executing your intentions.**

Finance, built around your intentions.

---

Intend is an autonomous financial agent. Users express financial intentions in natural language. Intend interprets the intention, builds an execution plan, confirms with the user, and executes across DeFi and payment infrastructure — handling all intermediate steps invisibly.

## Status

v0.5 — Active development. Base-only execution.

## Architecture

Turborepo monorepo. TypeScript strict mode. Node.js v22.

See `CLAUDE.md` for full architecture, design principles, and agent coordination protocol.
